## External Link Configuration

### Method 1: Direct Link

Use `<a>` tag in HTML to link directly to external website:

```html
<a href="https://jogosdehoje.eu.com/" 
   target="_blank" 
   rel="noopener noreferrer">
    Go to External Site
</a>
```

### Method 2: JavaScript Redirect

Use JavaScript for redirection:

```javascript
window.location.href = 'https://jogosdehoje.eu.com/';
```

### Method 3: Meta Tag Redirect

Use meta refresh tag in `<head>`:

```html
<meta http-equiv="refresh" content="3;url=https://jogosdehoje.eu.com/">
```

## Deploy to GitHub Pages

1. Push code to GitHub repository
2. Go to repository Settings > Pages
3. Select branch and folder (usually `main` branch `/root`)
4. Save, and GitHub Pages will automatically deploy

## Local Development

## Important Notes

2. Redirects may affect SEO, use with caution
3. Ensure external links are legitimate
